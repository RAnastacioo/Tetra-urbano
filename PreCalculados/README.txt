Na pasta “Porto” existem dados padrão pré calculados para as condições standard 
implementadas na ferramenta, qualquer alteração carece de novos cálculos.
Para usar estes dados basta copiar todos os ficheiros da pasta “Porto”
e colocá-los na raiz de programa, posto isto apenas é necessário correr o script “Main”

Apenas não foram pré calculados resultados para a modelação IEEE 802.16 B e C.
